<h1>Sistema de Acompanhamento de Humor - Motion</h1>

<h2>1. Introdução</h2>
<p>
O projeto consiste no desenvolvimento de um sistema de acompanhamento de humor integrado a uma assistente virtual, inspirado em aplicativos como o Daylio. 
O sistema permite que usuários autenticados registrem seus sentimentos, adicionem descrições e tags, e visualizem esses registros em um calendário.
</p>
<p>
O principal objetivo é promover autoconhecimento emocional, oferecendo apoio ao usuário de forma prática, organizada e personalizada.
</p>

<h2>2. Funcionalidades do Sistema</h2>

<h3>2.1 Autenticação</h3>
<p>
O sistema permite que o usuário crie uma conta e faça login. Todas as funcionalidades ficam vinculadas ao usuário autenticado, garantindo privacidade e personalização.
</p>

<h3>2.2 Registro de Sentimentos</h3>
<p>O usuário pode registrar seu estado emocional escolhendo entre os sentimentos:</p>

<ul>
  <li>Calmo</li>
  <li>Feliz</li>
  <li>Triste</li>
  <li>Bravo</li>
  <li>Desapontado</li>
  <li>Preocupado</li>
  <li>Assustado</li>
  <li>Frustrado</li>
  <li>Estressado</li>
</ul>

<p>
Além disso, pode adicionar um texto explicativo e selecionar tags como “trabalho”, “escola”, “família” ou “amigos”.
</p>

<h3>2.3 Formulário de Preferências</h3>
<p>O formulário de preferências coleta informações como:</p>
<ul>
  <li>Formas que o usuário prefere receber ajuda</li>
  <li>Atividades que ajudam em momentos difíceis</li>
  <li>Outras preferências relacionadas ao bem-estar</li>
</ul>
<p>Essas informações tornam o sistema mais personalizado.</p>

<h3>2.5 Calendário de Registros</h3>
<p>
O calendário exibe o humor predominante de cada dia por meio de cores. Ao clicar em uma data, o usuário pode visualizar:
</p>
<ul>
  <li>Notas do dia</li>
  <li>Sentimentos registrados</li>
</ul>

<h3>2.6 Notas</h3>
<p>O usuário pode criar notas adicionais para registrar informações que não estejam relacionadas diretamente a sentimentos.</p>

<h3>2.7 Perfil do Usuário</h3>
<p>A página de perfil permite visualizar e editar informações pessoais, contribuindo para uma experiência mais personalizada.</p>

<h3>2.8 Páginas de Erro</h3>
<p>Foram implementadas páginas de erro personalizadas para melhorar a experiência do usuário em caso de falhas.</p>

<h3>2.9 Banco de Dados</h3>
<p>
O banco de dados armazena:
</p>
<ul>
  <li>Dados dos usuários</li>
  <li>Registros de sentimentos</li>
  <li>Textos explicativos</li>
  <li>Tags</li>
  <li>Datas e horários dos registros</li>
</ul>
<p>Garantindo organização e segurança.</p>

<h3>2.10 Nora</h3>
<p>
A página da assistente virtual exibe inicialmente o humor registrado do dia e um botão para iniciar sugestões.
Após a interação, recomendações personalizadas são apresentadas no formato de mensagens, simulando uma conversa.
</p>
